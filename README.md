  - ### 차트 (일간, 주간, 월간) [🔗코드 보기](https://github.com/EllyKimHub/pandang/wiki)

3. 상품의 수량 최소 개수가 1 이상의 정수
4.  체크박스 선택하면 총 상품 금액, 총 배송비, 결제 예정금액 계산
5.  체크박스 전체 선택 사용
6.  체크박스 선택으로 상품 삭제 
7.  상품주문 클릭 시 결제하기 페이지로 이동
8.  체크박스가 선택이 안되어있는 경우 구매 페이지로 이동 불가
